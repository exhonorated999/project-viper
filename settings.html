Investigative Tools

These modules can be added to any case via the Add Module button on the case detail page.

CDR — Call Detail Records

Import and analyze call detail records from carrier subpoenas. Generates interactive timelines, network relationship graphs, and geolocation maps to visualize communication patterns between targets over time. Supports standard CSV/XLSX tower dump and CDR formats.

Aperture — Email Forensics

Parse and analyze email evidence from .mbox and .eml files obtained through search warrants or legal process. View full email threads with headers, body content, and attachments in a sandboxed environment. Filter by sender, date range, and keyword search to identify relevant communications.

Network Intelligence — WiGLE

Query the WiGLE wireless geolocation database to map WiFi access points, Bluetooth devices, and cell towers. Useful for placing a device at a location by searching BSSID (unique MAC address), or tracking movement patterns via SSID history. Requires a free WiGLE API account.

Configure API Credentials

Get your API credentials from wigle.net/account

Trace Import — TRACE Case Packages

Import case packages exported from TRACE (Targeted Retail & Cargo Intelligence Engine) to cross-reference suspects, vehicles, evidence, and case narratives from retail/cargo theft investigations. Imported data is read-only and preserved as reference material alongside your VIPER case.
