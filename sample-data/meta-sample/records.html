Unified Messages Definition
Unified Messages: Private communication between account holders.
Thread: Unique identifier associated with conversation thread.
Current Participants: Account holders included in conversation thread.
Author: Account holder that sent communication.
Sent: Date and time of communication sent.
Removed by Sender: Indicates that the communication has been removed by the sender.
Call Record: Indicates a call was made through conversation thread.
Type: Indicates the method in which a call was placed.
Missed: Indicates if the call was missed.
Duration: Length of the call (in seconds).
Thread Name: Name given to the chat thread by the group admin.
Encrypted Calls: Calls that were encrypted.
Attachments: Files associated with the communication. Type: Description of what type of attachment. Size: The size of the file sent. URL: URL of the content.
Body: Text of communication, or description of content type sent.
Group Profile Picture: The current profile photo of the group chat at the time of production.
Nickname: An optional alternate name for the account holder that can be created/updated by any participant in the specific thread.
Past Participants: Lists the account holders that have previously been subscribed to the conversation thread. Subscription Event: Indicates an update to a participant's subscription to the conversation thread. Type: Indicates action associated with the participant. Users: the account holders impacted by the subscription update.
Share: Items shared into the communication thread between account holders. Date Created: Date and time associated with the share. Text: Content of the share. URL: URL of the content.
