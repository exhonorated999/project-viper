Investigative Tools

These modules can be added to any case via the Add Module button on the case detail page.

Aperture — Email Forensics

Parse and analyze email evidence from .mbox and .eml files obtained through search warrants or legal process. View full email threads with headers, body content, and attachments in a sandboxed environment. Filter by sender, date range, and keyword search to identify relevant communications.

ARIN Lookup — IP WHOIS

Query the ARIN (American Registry for Internet Numbers) WHOIS database to identify the registered owner, ISP, and network allocation for a given IP address. Returns the organization name, network range, and CIDR block. Automatically follows redirects to other Regional Internet Registries (RIPE, APNIC, etc.) for non-North American IPs. Available on IP address identifiers in the Suspects module.

CDR — Call Detail Records

Import and analyze call detail records from carrier subpoenas. Generates interactive timelines, network relationship graphs, and geolocation maps to visualize communication patterns between targets over time. Supports standard CSV/XLSX tower dump and CDR formats.

Email Verifier — SMTP Validation

Verify whether an email address has a valid, deliverable mailbox by performing a real-time SMTP handshake (EHLO → MAIL FROM → RCPT TO) against the domain's mail server. Checks DNS MX records and validates the mailbox exists without sending an actual email. Some providers (Apple/iCloud, Yahoo, ProtonMail) block mailbox-level verification as an anti-spam measure — these are reported as "unverifiable" rather than invalid. Available on email identifiers in the Suspects module.

FMCSA — Carrier Lookup

Search the FMCSA SAFER (Safety and Fitness Electronic Records) database to look up motor carriers by USDOT number, MC/MX number, or company name. Returns carrier snapshots including safety ratings, operating status, insurance, vehicle/driver counts, and crash history. Useful for cargo theft investigations to identify and verify trucking companies, brokers, and freight forwarders involved in a case.

Configure API Key

Get a free API key from FMCSA Developer Portal. Registration is free and instant.

GenLogs — Cargo Intelligence

Integrate with GenLogs for carrier vetting, vehicle tracking alerts, and freight intelligence. Query DOT/MC numbers, monitor carrier safety scores, and receive real-time alerts on cargo movements relevant to your investigation.

Network Intelligence — WiGLE

Query the WiGLE wireless geolocation database to map WiFi access points, Bluetooth devices, and cell towers. Useful for placing a device at a location by searching BSSID (unique MAC address), or tracking movement patterns via SSID history. Requires a free WiGLE API account.

Configure API Credentials

Get your API credentials from wigle.net/account

Ping — Network Reachability

Send ICMP ping requests to an IP address to determine if the host is currently online and reachable. Displays round-trip latency and packet loss statistics. Useful for quickly confirming whether a suspect's server, IP camera, or networked device is active. Available on IP address identifiers in the Suspects module.

RMS Import — Agency Reports

Import and parse PDF reports exported from agency Record Management Systems (RMS). Automatically extracts structured sections including events, offenses, involved persons, vehicles, property, narratives, and confidential informants. Currently supports INFORM RMS format. Imported reports are displayed in a card grid with expandable detail views.

Trace Import — TRACE Case Packages

Import case packages exported from TRACE (Targeted Retail & Cargo Intelligence Engine) to cross-reference suspects, vehicles, evidence, and case narratives from retail/cargo theft investigations. Imported data is read-only and preserved as reference material alongside your VIPER case.

TRACE Network Search

Search the TRACE retail theft & cargo intelligence network for cross-agency matches against license plates, suspect names, MC numbers, and BOL/CDL numbers. Requires a full VIPER license.
